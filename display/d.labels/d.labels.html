DESCRIPTION

d.labels displays a paint label file in the active display frame on the graphics monitor. Each label has components which determine the text, the location of the text on the image, its size, and the background for the text. NOTES

Some of the information stored in the label file is unused by d.labels. This extra information is used by such programs as ps.map.

This module was formerly known as d.paint.labels. The the old version of d.labels from GRASS 5, which provided interactive placement and modification of paint labels, still needs to have its functionality merged into this module.
